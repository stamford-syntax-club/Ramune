# Stamford Syntax Club's Discord Bot 🤖

[![Discord.NET Version](https://img.shields.io/badge/Discord.NET-v3.12.0-blue.svg?style=for-the-badge)](https://www.nuget.org/packages/Discord.Net/)
[![.NET Framework](https://img.shields.io/badge/.NET%20Framework-6.0-blue.svg?style=for-the-badge)](https://docs.microsoft.com/en-us/dotnet/csharp/)

Welcome to the Stamford Syntax Club's Discord bot repository! Crafted with ❤️ using the Discord.NET library in C#, this bot will provide various club management tools, serves tunes 🎶, and sprinkle in some fun to the server. Ready to dive in? 🌊

## Features 🚀

1. **Club Management 📋**
   - Register new members
   - Announce upcoming events and meetings
   - Poll creation for group decisions
   - Track and reward participation

2. **Music Features 🎵**
   - Play, pause, skip, and queue songs
   - Volume control and track information
   - Create and manage playlists

3. **Fun & Games 🎲**
   - Random coding challenges
   - Quick quizzes
   - Meme generator for the coding enthusiasts

4. **AI Utilities** 🤖
   - Chat with GPT-4
   - Others TBD

## Getting Started 🛠

1. **Prerequisites**
   - .NET SDK 6.0
   - A Discord token

2. **Setup**
   1. Clone the repository:
   ```
   git clone https://github.com/StamfordSyntaxClub/DiscordBot.git
   ```
   2. Navigate to the project directory and install dependencies:
   ```
   cd DiscordBot && dotnet restore
   ```
   3. Set up a configuration file to provide your Discord token and the guildIds and whatnot.

3. **Running the Bot 🚀**
   ```
   dotnet run
   ```
